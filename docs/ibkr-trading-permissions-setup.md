# IBKR 复杂或杠杆交易所交易产品许可填写指引

## 目的

你需要让 IBKR 按真实经历完成适当性评估。虚报年限、交易次数或知识水平可能造成许可被撤销，也会让你承担不匹配的风险。

## 前置

准备各产品第一次真实交易的大致年份、近几年每年平均交易次数；确认能登录 Client Portal。模拟盘和课程学习不要算作真实交易，但可用于判断知识水平。

## 填写步骤

1. 进入「右上角头像 → 设置 → 交易 → 交易许可」，请求「复杂或杠杆交易所交易产品」。**Why：**这是 IBKR 当前管理产品权限的入口。
2. 股票、期权、期货分别填写实际交易年数和年均次数；从未交易就填 0 或界面提供的最低真实档。**Why：**三类产品不能互相借用经验，买过股票不等于有期权或期货经验。
3. 知识水平按能力选择：只懂基础选 Limited；能独立解释交易机制、保证金和主要亏损路径选 Good；长期频繁实践并理解高级策略才选 Extensive。**Why：**IBKR 官方说明，获准交易某产品通常需要 Good 或 Extensive，但这不是让你虚报。
4. 阅读并确认风险披露后提交；若出现测验，按自己的理解作答。**Why：**杠杆/反向 ETF 有每日重置与路径依赖，ETN 还有发行人信用和流动性风险。

## 成功标志

页面显示请求已提交、待审核或许可已启用；正常情况下，权限升级请求会接受监管审核，并非填写某组答案就保证通过。

## 常见卡点

- 经验不足：先学习 IBKR Campus、使用模拟账户，再按真实情况重新申请。
- 资料互相矛盾：核对投资目标、财务资料与风险承受能力，勿为过审改成不真实信息。
- 找不到产品：先确认所在 IBKR 实体、年龄、账户类型和地区是否支持。

## 填回到哪

这些答案只填在 IBKR Client Portal，不写入本仓库，也不要保存账号、密码或验证码。参考：[交易许可说明](https://www.ibkrguides.com/orgportal/tradingpermissions.htm)、[复杂/杠杆 ETP 风险披露](https://gdcdyn.interactivebrokers.com/Universal/servlet/Registration_v2.formSampleView?formdb=4853)。
