> 面向：想在中国大陆上架付费 App / 小程序的个人开发者核心问题：要不要备案？需不需要 100 万注册资本？个人能不能干？

------

## 一、核心概念区分

### 三种"备案"别搞混

| 类型                            | 对象                 | 是否收费                 | 个人能办    |
| ------------------------------- | -------------------- | ------------------------ | ----------- |
| ***\*ICP 备案\****（非经营性）  | 网站/域名            | 免费                     | ✅           |
| ***\*ICP 经营许可证\****        | 经营性互联网信息服务 | 需申请，注册资本 ≥100 万 | ❌ 仅限企业  |
| ***\*App 备案 / 小程序备案\**** | 移动应用 / 小程序    | 免费                     | ✅（有限制） |

### 关键认知：ICP 经营许可证管的是"信息服务"，不是"商品交易"

- ✅ 卖工具、SaaS、数字商品、付费功能 → ***\*只需 ICP 备案\****
- ❌ 付费阅读资讯、付费数据库、经营性内容平台 → ***\*需 ICP 经营许可证\****

------

## 二、注册资本 100 万的真相

- ICP 经营许可证要求"注册资本 ≥100 万"指***\*营业执照上的认缴金额\****，不用实际打款
- ⚠️ ***\*2024 年 7 月新《公司法》\****：认缴制保留，但 ***\*5 年内必须缴足\****
- 多数个人开发者***\*根本不需要 ICP 经营许可证\****，因此不需要 100 万

------

## 三、主体选择：个体工商户 vs 有限公司

| 维度           | 个体工商户             | 有限公司                   |
| -------------- | ---------------------- | -------------------------- |
| 注册成本       | 0–500 元               | 1000–2000 元               |
| 年度成本       | ~2800 元               | ~5000 元                   |
| 责任           | ⚠️ 无限责任             | ✅ 有限责任                 |
| ICP 备案       | ✅ 可办                 | ✅ 可办                     |
| 微信支付商户   | ✅ 可开                 | ✅ 可开                     |
| ICP 经营许可证 | ❌ 不能申请             | ✅ 可申请                   |
| 适合场景       | 工具类、SaaS、付费下载 | 长期发展、需融资、UGC 内容 |

***\*结论\****：付费 App 大多数情况下***\*个体工商户即可\****，无需公司+100 万注册资本。

------

## 四、关于平台支付（绕开 ICP 经营许可证的关键）

### 借平台资质 = 不需要 ICP 经营许可证

| 渠道                             | 抽成               | 资质要求           |
| -------------------------------- | ------------------ | ------------------ |
| Apple 内购（IAP）                | 30%                | 由 Apple 持牌经营  |
| 微信小程序虚拟支付（Android）    | 0.6%               | 借腾讯资质         |
| 微信小程序虚拟支付（iOS）        | ❌ 禁止，必须走 IAP | —                  |
| 知识付费平台（小鹅通、知识星球） | 平台规则           | 平台已持有所有牌照 |

> 💡 ***\*核心思路\****：你做"商家入驻"，让平台做"经营主体"。

------

## 五、个人 vs 个体户主体的能力差异

### 微信小程序主体能力（决定性差异）

| 功能         | 个人主体         | 个体户/企业主体 |
| ------------ | ---------------- | --------------- |
| 注册小程序   | ✅                | ✅               |
| 微信支付     | ❌ ***\*不能\**** | ✅               |
| 虚拟支付     | ❌                | ✅（Android）    |
| 类目可选范围 | 极少             | 全部            |
| 备案审核难度 | 严格             | 宽松            |

### 个人 ICP 备案的硬限制

- ❌ 网站名不能含"商城/交易/论坛/科技"等商业词
- ❌ 必须承诺"非经营性"，不能付费、不能挂广告
- ❌ 不能开通微信支付商户号
- ⚠️ 被举报会注销备案 + 1 年禁止再备

## 六、Cloudflare 域名 + 备案的坑

```
Cloudflare 注册的域名 → 工信部不认可境外注册商        ↓无法直接做国内 ICP 备案
```

***\*解决方案\****：

1. ***\*转出域名到阿里云/腾讯云\****（保留 DNS 仍可用 CF）
2. ***\*再买一个国内备案专用域名\****（~50 元/年）
3. ***\*纯海外路线\****（但小程序后端域名强制要求备案，此路对小程序行不通）

> ⚠️ ﻿﻿﻿.dev﻿ ﻿﻿﻿.app﻿ ﻿﻿﻿.io﻿ ﻿﻿﻿.ai﻿ 等部分新顶级域名国内不支持备案。

## 七、推荐方案（按需求）

### 🟢 方案 A：免费练手 / 验证产品

```
个人微信号注册小程序+ 个人 ICP 备案（备案描述写"个人学习记录"）+ 国内服务器= ~650 元/年，仅限免费功能
```

### 🟢 方案 B：⭐ 想赚钱（推荐）

```
注册个体工商户（淘宝代办 ~300 元）+ 个体户主体小程序 + 微信认证 300 元/年+ 个体户 ICP 备案 + 小程序备案+ 微信支付商户号= ~3800 元/年首年，可正常付费
```

### 🟢 方案 C：纯海外

```
Cloudflare 域名 + 海外服务器 + Stripe/IAP= 不备案、不上中国应用商店、放弃国内市场
```

## 八、备案完整流程要点

### ICP 备案（约 7–20 工作日）

1. 在国内云厂商买服务器（≥3 个月时长）
2. 拥有可备案后缀域名（且实名认证 ≥3 天）
3. 提交主体 + 网站信息 + 真实性核验（人脸或幕布）
4. 云厂商初审 → 通信管理局复审 → 下发备案号
5. 网站底部展示备案号 + 30 天内完成公安备案

### 小程序备案（约 7–20 工作日）

1. 注册小程序 + 完成认证（个体户/企业 300 元/年）
2. 后台「设置」→「小程序备案」入口
3. 关联已 ICP 备案的服务器域名
4. 微信平台初审 → 工信部复审 → 自动展示备案号

> 🔑 ***\*顺序\****：必须先做 ICP 备案，再做小程序备案。

------

## 九、关键避坑清单

- ⛔ 个人备案绝不能写"小程序后端 / API 服务 / 对外提供"
- ⛔ 个人主体小程序不能加付费功能（违规即封）
- ⛔ Cloudflare 域名直接备案 → 不可能，先转出
- ⛔ 备案期间网站不能上线（会被检测驳回）
- ⛔ 注册资本不要盲目写 100 万（5 年内要实缴）
- ✅ 备案审核员会电话回访 → 保持手机畅通
- ✅ 备案号下来后 30 天内必须做公安备案

## 十、最终决策建议

```
你的目标？│├─ 只想做免费小程序练手│   └─ 个人主体 + 个人 ICP 备案 即可│├─ ⭐ 想做付费应用赚钱│   └─ 注册个体工商户 是唯一合规且可行的路径│       理由：│       • 个人主体小程序无法开通微信支付（硬规定）│       • 个人 ICP 备案禁止经营性业务│       • 个体户成本约 3000 元/年，远低于公司│├─ 业务复杂 / 需要融资 / 涉及 UGC│   └─ 注册有限公司，必要时申请 ICP 经营许可证│└─ 不想搞备案    └─ 纯海外路线（放弃国内市场）
```

## 📌 一句话总结

> ***\*想合法做付费应用赚钱，注册个体工商户是绕不开的最低成本路径\****。个人 ICP 备案能做但不能用于经营；100 万注册资本不是必须的，多数业务靠"借平台资质"（Apple 内购、微信小程序）就能合规收钱。
