## 二、套到 howto-make-more-money 上,排个优先级

### 立刻能做(低风险、跟主业互相反哺)

#### 1. Unity 编辑器扩展 / 工具插件 — Asset Store 或国内市场

你那个"爆框检测"工具就是现成种子。所有用 Unity 做大量 UI 的项目都需要,**痛点确定 + 你已经做过 6 个版本**。

- 打磨一下出英文 README → 上 Asset Store
- 定价 \$15-30 不难
- 再做"策划表 → ScriptableObject 自动生成 + diff 校验"姊妹工具,一条小产品线就起来了

→ 对应 howto"做产品"分支里**门槛最低**的那一条:不用做美术、不用做服务器、用户付费意愿明确。

#### 2. 接 Unity 编辑器 / Shader 小活

不是接整个项目外包,而是 200-2000 元一单的小活:

- 渠道:猿代码、QQ 群、小红书、独立开发者社群
- 内容:URP 升级、Shader 某个效果、编辑器小工具
- 你的工具链经验是溢价点

---

### 半年到一年纬度(要持续投入,但能复利)

#### 3. "Unity 客户端 → TA 转型实录"内容号 — 知乎 / B 站 / 小红书

这是个**正在大规模发生的真实需求**:很多 Unity 程序员焦虑想转 TA,但市面上要么是高价培训班,要么是零散教程。

你有三个稀缺优势:
- 你正在路上(过来人 vs 局外人之差)
- 你已经把 Games101 / LearnOpenGL / Hazel 笔记写下来了,**内容现成**
- 你有"系统整理"的笔记习惯,适合长内容

起号路径:

```
每周 1 篇 Games101 / Shader 实战笔记
        ↓ (半年)
攒粉 → 知识星球(99-199/年) / 付费小课
```

→ 对应 howto"写作"+"教学"双线。

#### 4. Hazel 中文跟做仓库 + 系列博客

TheCherno 的 Hazel 在国内有需求但中文资料断断续续。你正在跟做,顺手把每一集的:

- 中文注释 + 补充推导整理成系列(知乎 / 掘金专栏)
- GitHub 中文注释仓库

容易吃到这波 SEO 流量,然后接到 #3 的内容号上。

### 中期机会(押 AI × 你的领域,出海概率更高)

#### 5. AI × Unity 编辑器小工具

你的工具链经验 + AI 绘图笔记都能用上。可做方向:

- "自然语言改 Shader"
- "一键生成法线贴图 / PBR 套图"
- "策划表 AI 自动校验"

出海路线:Asset Store / Gumroad / Patreon。比国内市场愿意付费、客单价高。

→ 对应 howto"出海"分支。**但你现在还不用急**,等 #1 把"做小产品"的肌肉先练出来再切。

---

### 不建议踩的坑

| 坑 | 原因 |
|---|---|
| 纯独立游戏发行 | 国内版号、海外发行经验你都没有 |
| 企业站 / 小程序外包 | 跟你技能栈错位,单价低还耗时 |
| AI 音乐 / 电音变现 | 笔记里只是兴趣,没积累到能卖东西的程度 |
| howto 里的 SEO 黑帽 / 刷量 / 私域套路 | 灰色 + 不可持续 |
