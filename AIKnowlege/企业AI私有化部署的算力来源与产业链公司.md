# 企业 AI 私有化部署:开源模型差距、算力来源与产业链公司

> 2026-07 与 Claude 的讨论整理,承接《AI烧钱竞赛、AI是否真有用与普通人如何上车》。数据以 2026 年初为准,价格看量级不抠零头。

## 一、中国开源模型和 OpenAI/Claude 差距大吗

- **纸面(benchmark)已经不大**:DeepSeek、Qwen3、Kimi K2 Thinking、GLM 在数学、代码、知识类基准上追到 GPT-5 / Claude 同代的 90%–100%,个别 agentic 基准甚至赢过。差距从 2023 年的约两年缩到三到六个月。
- **体感还有一段**:最硬的场景(agent 编码,连续几十步改代码、跑测试、自我纠错)闭源第一梯队仍明显领先。差距不在"答对题",在长任务可靠性:指令遵循稳定度、工具调用出错后的自我恢复、几十万 token 上下文不迷路。这些不太进 benchmark,但决定能不能托付真实工作。
- **开源阵营内部**:中国模型已是全球开源绝对主力。Llama 掉队后 HuggingFace 下载榜前列基本被 Qwen、DeepSeek 系包揽,海外创业公司拿中国开源模型做底座是常态。

**多久打平**:对"企业私有化部署够不够用"这个标准,**现在就已经打平**——客服、文档、知识库、常规代码辅助用 Qwen/DeepSeek 级开源模型完全够。但"最前沿那一步"会持续滞后 3–12 个月:芯片管制卡住训练算力上限,新范式总是算力最多的实验室先趟出来,别人再用十分之一成本复现。只要管制在,追赶者位置是常态。不过对私有化这门生意,"够用即打平"已成立,不用等。

## 二、私有化部署需要什么硬件

"算力"更准确说是**推理算力**(inference)——只运行模型不训练,门槛低得多。核心约束一句话:**模型权重必须装进 GPU 显存**。

| 模型档位 | 显存需求 | 对应硬件 | 大致成本 |
|---|---|---|---|
| 32B 级(Qwen3-32B) | 量化后 20–40GB | 1–2 张 RTX 4090/5090 | 几万元 |
| 70B–100B 级 | 80–200GB | 4–8 卡服务器 | 几十万元 |
| 旗舰 MoE(DeepSeek 671B、Kimi K2 1T) | 量化后 600GB+ | 8 卡 H200/B200 或昇腾服务器一到两台 | 一两百万元级 |

单机或两机推理不需要建"机房",一个标准机柜位放得下。国内政企市场还有更省事的形态:**大模型一体机**(华为、浪潮等把昇腾卡+DeepSeek/Qwen 预装好整机卖),开箱即用,2025 年在政企卖爆。

## 三、自建还是租云?一个关键矛盾

"自建太浪费所以租云"对一半。注意隐含矛盾:**企业私有化的动机就是数据不能出门**。如果数据可以上云,最便宜的方案根本不是租 GPU,而是直接调 API。真实世界分三档:

1. **数据无所谓** → 直接用 API,没有部署这回事,最便宜。
2. **数据敏感但接受专属资源** → 租云厂商**专有云/专区**(物理隔离的独占 GPU 资源),"租"和"私有"的折中,最大的中间市场。
3. **数据绝对不出门**(银行核心、政务、军工) → 买服务器/一体机放自己机柜或托管 IDC,"浪费"是合规成本。

纯经济账:租一张 H100 约 2–3 美元/小时,8 卡机年租十几万美元,买一台约 25 万美元加运维。**利用率超一半、用三年以上,买划算;间歇用,租完胜。**

## 四、500 人企业成本测算

场景假设:内部知识库+文档处理+部分人编码助手,高峰并发 30–50 路,跑 Qwen3-235B / DeepSeek 量化档,一台 8 卡推理服务器够用。

| | 自建(买服务器) | 租云 GPU(独占实例) | 调开源模型 API(对照组) |
|---|---|---|---|
| 前期投入 | 8×H20 约 ¥120 万 + 部署 ¥10 万 | 0 | 0 |
| 每年固定支出 | 托管+电费 ~¥10 万,运维 0.5 人力 ~¥15 万 | 包月 ¥5–8 万/月 → ¥60–95 万/年 | 按量付费 |
| 折算年成本(4 年) | **~¥55–60 万/年** | **~¥60–95 万/年** | **~¥10–30 万/年** |

数字来历:8×H20 整机 ¥100–140 万(昇腾一体机 ¥80–120 万),4 年折旧年均 ~¥30 万;整机满载约 10kW,电费+制冷+机柜托管年 ~¥10 万;API 对照按 500 人重度使用(日总量 2–3 亿 token、DeepSeek 级价格 ¥1–3/百万 token)年 10–30 万。

**结论三条**:

1. 数据允许出门,前两条路都不该走——API 便宜一个量级。私有化本质是花 30–80 万/年买"数据不出门",不是省钱手段。
2. 确定私有化,买还是租看回本周期:自建比租每年省 40–50 万但先垫 130 万,约 2.5–3 年打平。稳定用 3 年以上买,试点期租。常见路径:租 3–6 个月试点 → 采购一体机落地。GPU 租价一直在跌、硬件 3 年过时,时间对"租"有利,决策线要比纯算术保守。
3. 真正贵的不是服务器,是**长期养这套东西的人**。开源模型每季度换代,运维实际往往滑向 1–2 个人力。一体机和专有云托管在国内卖得好,企业买的不是硬件是"不用自己养人"。

## 五、租云算力,数据算不算出门?

**物理上出门了,但性质和调 API 完全不同**,信任边界分三档:

1. **调 API**:数据进入模型厂商的服务层处理,合规部门最过不去的一档。
2. **租云 GPU 自己部署**:权重、推理服务、数据全在独占实例/VPC 里,云厂商只提供裸算力。数据物理在云厂商机房,信任边界变成"自己+云厂商物理管控"。等保和多数行业监管接受,签数据处理协议+全程加密。
3. **本地/专属机房**:唯一满足"数据不出域"硬性监管的一档。折中形态:云厂商把机柜搬进企业楼里运营(专有云)。

一句话:监管要求"数据可控"→ 租云行得通;写明"不出域"→ 只能自建或一体机。

## 六、产业链受益公司(国内外)

> 常见混淆:**SMIC 是中芯国际,中国公司**,芯片代工;**SMCI 是 Super Micro(超微电脑),美国公司**,AI 服务器整机。两家都在链上但位置完全不同。

| 环节 | 全球龙头 | 中国龙头 |
|---|---|---|
| AI 芯片 | 英伟达(独占大头)、AMD、博通(定制 ASIC);代工全在台积电;HBM:SK 海力士/三星/美光 | 华为昇腾(未上市)、寒武纪、海光信息、摩尔线程、沐曦;代工中芯国际 |
| 服务器/机柜集成 | SMCI、戴尔、HPE;GB200 机柜主要由台系代工:鸿海、广达、纬创 | 浪潮信息(份额第一)、工业富联(英伟达机柜代工)、华为、新华三(紫光股份)、联想、超聚变(未上市) |
| 一体机 | 戴尔/HPE"AI Factory"、英伟达 DGX,海外不算大赛道 | 中国特色赛道:昇腾生态的神州数码、拓维信息、软通动力,加浪潮、新华三 |
| 云算力出租 | AWS、Azure、谷歌云、Oracle;新势力 CoreWeave、Nebius、矿商转型(IREN 等) | 阿里、腾讯、百度、字节(未上市);三大运营商;IDC:万国数据、世纪互联、润泽科技 |
| 机柜配套 | 液冷 Vertiv、交换机 Arista、光模块 Coherent | 光模块:中际旭创、新易盛(即全球龙头);液冷英维克、PCB 沪电/胜宏、存储模组江波龙/佰维 |

### 结构性判断

- **利润不均匀,链条位置决定赚不赚钱**:英伟达一家拿走整条链一半以上利润,毛利 70%+;服务器集成是苦生意,浪潮/工业富联毛利 5–10%,收入暴涨利润不怎么涨。"受益"和"赚钱"是两回事。配套里光模块例外——格局好毛利高,中际旭创、新易盛表现远超服务器厂。
- **一体机是渠道生意不是技术生意**:神数、拓维赚"华为生态代理+集成服务"的钱,门槛在渠道不在技术,题材属性大于业绩属性。
- **云算力出租的分化**:海外 neocloud(CoreWeave 等)是高杠杆重资产,赌租金回报跑赢 GPU 贬值,模式未被完整证明;国内智算中心大量政府/运营商主导,利用率参差,商业化健康的还是头部大云。
- **"500 人企业买一体机"场景的受益顺序**:芯片厂(昇腾/寒武纪)拿大头 → 整机厂(浪潮/华为)拿中间 → 渠道集成商(神数/拓维)拿服务费。
- **泼冷水**:这些 A 股标的人尽皆知,估值普遍把两三年预期打满。"产业确定受益"和"现在买入能赚钱"之间隔着估值,链条图帮你判断谁真受益,买卖时点是另一门功课。不构成投资建议。

## 七、超级个人/小工作室怎么选(附带结论)

九成以上不会本地部署,算力来源是云和 API,且主要是 API。三笔账:

1. **利用率账**:个人负载间歇性,买卡利用率不到 10%,API 纯按量。没有公司自己发电,除非是电解铝厂。
2. **贬值账**:token 价格每年跌一个量级,硬件在贬值,API 在降价,时间站在租用这边。
3. **能力账**(最易忽略):超级个人的杠杆来自用**最强模型**,最强模型永远先出现在 API 上。本地跑开源省的钱,远小于用次一档模型损失的产出。

例外:客户强制数据不出门、批量任务大到 API 账单超硬件摊销、要微调小模型、纯爱好(512GB Mac Studio 已能跑 DeepSeek/K2 量化版,但那是玩具和备胎,不是生产主力)。

终局图景:大企业按敏感度在"API—专有云—一体机/自建"三档分层;个人和工作室默认 API;算力走向公用事业化——像电一样,插插座的人不关心电厂在哪,发电厂(云厂商)和电网设备商(芯片、光模块、存储)是投资主线。
