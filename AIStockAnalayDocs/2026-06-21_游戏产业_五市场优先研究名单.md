# 游戏产业 · 五市场优先研究名单（Serenity 风格）

> 生成日期：2026-06-21
> 适用市场：美股、韩股、A 股、日股、欧股
> 每市场上限：5 家
> 性质：研究优先级排序与产业链卡点判断；非买卖建议，交易决策由读者自行负责。

## 1. 先看产业链卡点（这是研究的前置）

游戏产业的钱不会平均分给每一层。当下（2026 年中）真正稀缺的，按优先级是：

| 排序 | 卡住的环节 | 为什么稀缺 |
|---|---|---|
| 一线 | **顶级 IP 持有方 + 自研 AAA 双轨能力** | GTA / 任天堂第一方 / 宝可梦 / 怪物猎人 / Witcher / Cyberpunk 这种 IP 是复利的源头，跨代次、跨媒介、跨地区都吃。能反复生产 AAA 的团队全球不超过 20 家。 |
| 一线 | **平台 / 分发 + 自有第一方** | Nintendo Switch 2、Sony PS、Microsoft Xbox/Game Pass、苹果/谷歌商店。下游议价权 + 抽成（30%）+ 自有内容双叠加。 |
| 二线 | **「常青服务型游戏」运营 + IP 全球化能力** | PUBG / Genshin / Roblox / Brawl Stars / Marvel Rivals。少量爆款养整家公司、收入跨年稳定。 |
| 二线 | **AI / 广告变现 + 手游卖铲人** | AppLovin AXON、字节穿山甲、Tencent 广告。手游开发者越多、买量竞争越激烈，平台越赚。 |
| 三线 | **垂直精品研发 + IP 衍生力** | Pearl Abyss、Capcom、CD Projekt、米哈游、Larian、FromSoftware。靠"一代爆款定 3-5 年"。 |
| 排序靠后 | 纯渠道发行、纯 F2P 长尾、缺新 IP 的二线发行 | 议价权弱、被买量和平台双向挤压 |
| 排序靠后 | Unity 引擎本体（U） | 已被 UE5 / 自研引擎挤压，议价权下滑，不再是当年的卖铲人 |

**结论**：优先看「顶级 IP 持有方 + 自研 AAA」和「平台型大厂」。其次是有清晰常青产品的精品研发室。纯渠道 / 纯买量手游放最后。

## 7. 市场可能没看清的地方

- **A 股版号已经不是主要矛盾**：2025 全年 1771 个、2026 月度稳发，市场还在用版号叙事讲故事，但真正的差异已经转向"出海 + IP + 研发产能"三个维度。
- **EA 私有化（PIF / Silver Lake / Affinity Partners $55B、$210/股、预计 2026/6/30 前关闭）**：抽走了美股一个流动性极佳的"AAA 游戏纯标的"，剩下的纯游戏供给变得稀缺，**Take-Two 是最直接受益者**。
- **Krafton 印度 BGMI**：印度市场结构性增长被低估，付费用户 +17%，但海外卖方研报仍只把它当"PUBG 续作公司"。
- **Capcom + Konami 重制经济学**：自有 IP 全数字化重制毛利极高（Capcom 运营利润率已到 38.5%），市场仍按"日本传统游戏厂"估值。

## 8. 反方理由（什么情况说明这个判断错了）

- **GTA 6 第三次延期** —— TTWO 估值结构会崩。
- **Switch 2 销售在 FY27 显著低于 1650 万台指引** —— Nintendo 二年期的"硬件红利"故事破灭。
- **PUBG 印度遭遇政策反复** —— Krafton 单 IP 风险变现。
- **Crimson Desert DLC 上线效果差** —— Pearl Abyss 回归 2025 年那种亏损节奏。
- **A 股出海手游被海外平台抽成进一步上调或 SLG 红海化加剧** —— 三七、神州泰岳逻辑动摇。
- **CD Projekt 再发生类似 Cyberpunk 2077 的发售翻车** —— 波兰 AAA 信誉再损一次就是品牌坍塌。
- **EA 私有化交易因 CFIUS 审查延期或被否** —— 市场对游戏并购的风险定价重新走高。

## 9. 下一步要查的硬证据（按优先级）

1. **Take-Two FY27 单季营收 / 递延收入指引** + GTA 6 预购数据（数字版 / 实体版分布）。
2. **Krafton inZOI 主机上线日期** + 中国 / 印度区域营收拆分（10-Q）。
3. **Pearl Abyss Crimson Desert 单用户 ARPU + 主机 vs PC 分布** + DLC 路线图时间表。
4. **吉比特 / 三七 2026Q1 财报**（季报披露窗口 4 月底），重点看流水递延和销售费用率。
5. **Nintendo Switch 2 软件附着率**（FY26 软件 4871 万 / 硬件 1986 万 = 2.45×）的可持续性 + 第三方 IP 上线节奏。
6. **Capcom 数字下载占比 + 重制系列单作毛利率拆解**（年报附注）。
7. **CD Projekt Witcher 4 与 Cyberpunk 2 的人头 / 月研发成本** + 现金储备覆盖年数。
8. **Ubisoft Vantage Studios 拆分方案细节 + Tencent 持股比例**。
9. **EA 私有化 CFIUS 审查进度**（影响游戏 M&A 市场氛围）。
10. **A 股 4 月以后版号细分**：判断头部公司过审产品占比是否升高、行业集中度方向。

## 10. 一句话总结（按"研究优先级"，不是买入命令）

> **最值得优先研究的三大主线**：
> ① **TTWO（GTA 6 发售周期，FY27 营收基线重置）**
> ② **Krafton + Pearl Abyss（韩股 IP 单作驱动 + 印度结构性增长）**
> ③ **Capcom + Konami + Bandai Namco（日股自有 IP 数字化重制经济学）**
>
> A 股选 **吉比特 / 三七** 看出海与精品执行力；欧股的核心是 **CD Projekt** 的 AAA 长周期赌注。
>
> 交易决策永远在你手上——上面这套排序只是研究路径。
