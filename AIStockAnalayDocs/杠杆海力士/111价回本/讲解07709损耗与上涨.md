## 一、我的损耗知识从哪来

这不是网上随手抄的，也不是那些财经号的简化版。杠杆ETF的"波动率损耗"是**量化金融里的标准结论**，有明确的数学推导和学术文献（比如 Cheng & Madhavan 2009 关于杠杆ETF路径依赖的研究）。我可以自己把公式推出来验证，而不是转述别人的话。

而且坦白说——**那些财经文章给的"即使打平也亏损"的例子是对的，但不完整、甚至有误导性**。它们只讲了震荡市的坏处，故意不讲趋势市的好处。你看到"上涨可以覆盖损耗"的说法，**在特定条件下是正确的**，下面我用数学告诉你到底在什么条件下成立。

## 二、损耗的真相：它不是"必然亏损"，而是"波动的税"

2倍日杠杆产品的近似公式是：

```
杠杆基金收益 ≈ (标的累计涨幅)² × exp(−σ²·t)
```

拆成两部分看，你就明白为什么两种说法都对又都不全：

**1. 平方项 `(标的涨幅)²` —— 这是趋势市的"顺风"**

如果海力士**平滑单边上涨**，2倍日杠杆的复利是帮你的，不是害你的。举例：标的平滑涨50%（因子1.5），基金理论上涨到 1.5² = 2.25 倍，也就是 **+125%**。注意——标的只涨50%，2x基金却涨了125%，远超"2×50%=100%"。这就是"上涨能覆盖损耗"那派人说的道理，**在单边趋势里他们是对的**。

**2. 衰减项 `exp(−σ²·t)` —— 这是波动的"税"**

σ 是标的的波动率，t 是持有时间。**税率正比于波动率的平方**。这就是关键：
- 大盘指数 σ≈15%，σ²≈2.25%，一年损耗才2%出头，几乎可忽略——所以指数型2x ETF长期拿问题不大。
- **海力士是单一个股，年内涨273%、全韩杠杆最挤的票，σ大概在50%~70%**。取σ=60%，σ²=36%。**这意味着仅"波动"这一项，一年就吃掉约30%+**。

**所以你和那些人都对了一半：损耗到底严不严重，完全取决于海力士这半年是"平滑上涨"还是"剧烈震荡上涨"。** 海力士的问题恰恰是——它波动极大，即便方向对，路径也大概率是剧烈颠簸的，那个 σ² 的税会实打实地收。

## 三、把你的200目标算清楚

从现在90到200，是基金 **+122%**（因子2.22）。用上面的公式反推，海力士正股需要涨多少：

| 路径假设                          | σ²·t（半年）     | 标的需要涨   |
| --------------------------------- | ---------------- | ------------ |
| **理想：几乎无颠簸的平滑上涨**    | ≈0               | 约 **+49%**  |
| **现实：中等震荡的上涨（σ=60%）** | 0.18，衰减≈0.835 | 约 **+63%**  |
| **颠簸剧烈的上涨（σ=80%）**       | 0.32，衰减≈0.73  | 约 **+70%+** |

`推导：基金因子2.22 = 标的因子² × 衰减 → 标的因子 = √(2.22/衰减)`

**读懂这张表你就有答案了：**
- 好消息：如果你真判断对了、海力士到年底趋势性再涨50%~65%（考虑到它年内已涨273%、目标市值2万亿美元的多头逻辑，这不是天方夜谭），那么 **200 是能到的，损耗确实会被上涨吃掉**。你看好的人说的对。
- 坏消息：这有个隐含前提——**路径不能太颠簸**。如果海力士像最近ADR上市这样反复暴涨暴跌（你自己都观察到了），同样是涨到年底某个价位，2x基金拿到的会比"理想平滑"少一大截。而且半年是很长的持有期，对一个"官方写明不宜持有超过一日"的产品来说，你是在长期逆着产品设计初衷用它。

## 四、我的诚实结论

1. **"损耗必然吃掉收益"是错的**——单边牛市里杠杆复利是你的朋友，这点那些看多的人没说错，我之前几轮也应该讲得更平衡。
2. **"损耗不用担心"也是错的**——因为海力士波动率极高，σ² 那个税率是大盘指数的十几倍，颠簸市里它会凶狠地磨你。
3. **两者的真正分歧点是"路径"，不是"方向"**。你赌的其实是两件事同时成立：①海力士年底真涨50%+（方向）②这半年别太颠（路径）。第二个条件对海力士这种票是最不确定的。

**如果你铁了心看多到年底200**，一个更聪明的做法是：用 **07709 表达短期弹性 + 用 SKHY(ADR)或韩股正股打底长期敞口**。正股/ADR是1倍、无损耗、可以真正"等到年底"；07709只在你判断某段趋势要爆发时加上去博弹性。这样你既拿到了单边趋势的平方级放大，又不用在颠簸期被 σ² 长期收税。纯靠 07709 单一工具扛到年底，等于把"方向对"和"路径顺"两个赌注绑死在一起，赢面反而变窄。

（以上是产品机制和量化推导，不构成投资建议；σ 的具体取值请以海力士实际历史波动率为准，我给的是量级估算。）
