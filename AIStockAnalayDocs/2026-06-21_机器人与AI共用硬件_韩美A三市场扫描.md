# 机器人产业与 AI 产业共用硬件 / 跨市场公司扫描

> **生成日期**：2026-06-21
> **方法**：Serenity 风格深度调研 —— 从系统变化 → 共用底层 → 卡点排序 → 跨市场公司清单 → 反方理由 → 下一步验证
> **市场范围**：韩股、A 股、美股
> **公司数量上限规则**：每个共用硬件 × 每个板块 ≤ 5 家（max(0, 5)）
> **性质**：研究优先级建议，**非买入建议**；最终决策与仓位由使用者负责

## 一、先把"共用"想清楚：机器人和 AI 真正交叠的底层

机器人产业 ≠ AI 产业。机器人独有的部分（精密减速器 RV / 谐波、稀土永磁伺服电机、力矩 / 触觉传感器、轻量化关节本体）和 AI 独有的部分（数据中心冷却、超算交换机、CPO）都先排除。

**真正"两边都不可或缺"的底层共用环节**有 5 类，按"扩产难度 + 替代难度"排：

| 共用环节 | 在 AI 里的角色 | 在机器人里的角色 | 卡点强度 |
|---|---|---|---|
| ① **AI 计算芯片（GPU / 边缘推理 SoC / NPU）** | 训练 + 数据中心推理 | "大脑"：VLA 模型推理、规划、视觉 | 极强（设计端高度集中） |
| ② **高带宽内存（HBM）+ 高速 DRAM / LPDDR** | 训练带宽刚需 | 端侧大模型推理（Jetson Thor 已上 LPDDR5X，下一代会上 HBM） | 极强（仅 3 家） |
| ③ **功率半导体（SiC / IGBT / GaN）** | 数据中心 PSU、48V 母线、HVDC 转换 | 关节电机驱动、电源管理、电池管理 | 强（高压 SiC 6 寸 → 8 寸瓶颈） |
| ④ **CMOS 图像传感器 (CIS)** | 训练数据采集、AI 视觉模型生态 | 视觉感知最核心组件，多目 + 鱼眼 | 中-强（高端 CIS 寡占） |
| ⑤ **高端 PCB / IC 载板（HDI、ABF 载板）** | AI 服务器主板 + 加速卡载板 | 控制板、Jetson 模组载板 | 强（ABF / M8 级以上紧缺） |

> 卡点排序的逻辑：HBM 的扩产周期 18-24 个月、SiC 8 寸良率仍低、ABF 载板 Ajinomoto 树脂膜全球独家、CIS 高端基本三家——这些都是"系统改了，但供给爬不上去"的环节。AI 芯片设计端集中度高，但 AI 芯片本身的产能瓶颈实际在 HBM、CoWoS 和 ABF 上。

## 三、跨环节优先研究名单（如果只盯几个最关键的）

按"两边都不可或缺 × 供给最难扩 × 替代最难"综合排：

1. **HBM**（韩股 SK Hynix / Samsung，美股 Micron）—— 卡得最死的一层
2. **AI 计算芯片**（美股 NVIDIA、A 股 寒武纪 / 瑞芯微）—— 价值量最大
3. **高端 PCB / ABF 载板**（韩股 ISU Petasys / Samsung EM、A 股 沪电 / 深南）—— 最容易被市场忽视的"管道"
4. **SiC / 功率器件**（美股 ON / MPWR、A 股 斯达半导）—— 机器人量产爆发期边际弹性大
5. **CMOS 图像传感器**（A 股 韦尔股份、韩股 三星）—— 视觉刚需，但单价不高

## 四、市场可能没看清的地方

- **HBM 不会只属于"训练"**：人形机器人下一代主控（≥2026 年的 Thor 后续）、自动驾驶 L4 域控很可能上 HBM。当前市场只把 HBM 算作 AI 数据中心受益。
- **ABF 载板**：AI 加速卡用量年化增长 30%+ 的情况下，全球只有 4-5 家能做高端规格，且 Ajinomoto ABF 膜独家。市场对芯片设计公司炒得很热，但载板供给侧的紧度被低估。
- **机器人电源是 SiC 真正放量的下一站**：当前 SiC 行情主要由车载驱动，机器人关节驱动器 + 数据中心 HVDC 是接力点。
- **国产 CIS（韦尔 / 思特威）在机器人视觉的份额**：远比市场认知的高，机器人厂商常用 OmniVision / 思特威方案，不是索尼独占。

## 五、反方理由 / 最大风险

| 风险 | 触发条件 |
|---|---|
| **HBM 价格周期反转** | 三星 HBM4 大规模量产 + Micron 良率追上 → 紧缺缓解 → 韩股利润见顶 |
| **AI 资本开支放缓** | 超大云厂商 capex 季度环比转负 → PCB / 载板 / HBM / 功率链全链回撤 |
| **人形机器人量产节奏低于预期** | 2026-2027 年特斯拉 / Figure 量产数 < 市场预期 → 机器人独享逻辑减弱，只剩 AI 单边 |
| **Wolfspeed 类供给侧事件** | SiC 龙头财务出问题 → 供给收缩对二三线供应商短期利好但行业信心受挫 |
| **国产替代节奏不及预期** | 美国出口管制升级 → A 股 AI 算力芯片估值压制；同时国内厂商技术追不上 → 双杀 |
| **估值已经透支** | NVIDIA、SK Hynix、ISU Petasys、沪电、寒武纪当前估值已含较多预期，需用增量数据持续兑现 |

## 七、几句必要的边界

- 上面是**研究优先级**而非买入建议；公司是否值得买，要结合估值、仓位、风险承受度自己决。
- 部分判断（如 HBM 进入机器人时间表、韦尔在机器人份额、ISU Petasys 客户集中度）是**中等强度证据**，需要按上表去翻原始公告才能升级到强证据。
- 韩股个别中小盘流动性较薄，下单时点和规模需要单独评估，不在本扫描范围内。
- 本扫描截止 2026-06-21；超过 1-2 个月后请重跑业绩、估值和供需位置。
