# AI 产业未来 3 年最被低估的环节 —— 全市场 Serenity 扫描

> 生成时间：2026-06-21
> 范围：A 股 + 美股 + 港股 + 台股 + 日股 + 韩股
> 时间窗口：1-3 年（中期，覆盖 HBM4 / CPO / Rubin / 玻璃基板代际切换）
> 方法：Serenity 产业链卡点分析法（市场叙事 → 系统变化 → 卡点层级 → 公司映射 → 证据 → 反方 → 下一步）
> 性质：**初步研判**（initial pass），关键数据点需用财报/官方公告验证
> 风险边界：研究判断而非交易指令。最终决策权在用户。

## 一句话结论

**未来 3 年最被低估的不是算力卡，而是"AI 算力能不能真正落地"的物理约束层——电力配电（HVDC + 变压器）、先进封装的后段（玻璃基板 / HBM4 / SoIC / TC Bonder）、液冷供应链、HBM 后段测试与关键耗材。**

这几层共同特点：
- 扩产周期 2-5 年，几乎是物理常数
- 软件不能绕过
- 供应商高度集中（往往 2-3 家寡头）
- 市场注意力集中在"算力卡 + 通用 HBM + 通用光模块"上，定价饱满
- 真正卡产能爬坡的环节没被充分定价

## 市场可能没看清的地方

1. **算力故事最终被"电"和"热"卡住**——超大规模云厂商订单出去了，但变电站和液冷设备能不能跟上还没充分定价
2. **CoWoS 之后会切换到 SoIC + 玻璃基板**——市场还在炒 CoWoS 受益方，2026-2027 的代际切换会重新洗牌
3. **HBM 良率会比想象更差**——HBM4 16-Hi 堆叠的良率挑战导致测试时间倍增，测试机+耗材层有量价齐升空间
4. **A 股的电力 / 液冷供应链不是"映射"是"实质受益"**——国内数据中心 capex 2026-2027 真要上量，金盘、英维克、沪电这条线兑现度被低估
5. **韩美半导体的卡点强度被普通投资者低估**——HBM 故事讲到 SK 海力士却不知道给它供 TC Bonder 的人，研究链条断了
6. **北美变压器的真受益者不在美国本土**——韩日变压器企业向美国出口才是物理稀缺的来源

---

## 什么情况说明这个判断错了

- **GPU 需求曲线塌陷**：如果大模型 scaling 收益边际明显下降，超大规模云厂商砍 capex 25% 以上 → 电力 / 液冷 / 封装全部估值杀
- **能效大幅突破**：如果新一代 GPU 能效提升 3x（不是 1.5x），液冷和配电压力部分缓解
- **玻璃基板被推迟到 2030+**：Corning / SKC 受益时间窗后移
- **国产替代失败**：A 股映射股（金盘、英维克、沪电、安集）需要看到客户验证证据
- **地缘冲突**：台股 / 日股 / 韩股的 TSMC、爱德万、Disco、SK 海力士估值被地缘 discount 长期压制
- **HBM 良率快速突破**：测试机和耗材层量价提升不及预期
- **TC Bonder 路线被三星自研替代**：韩美半导体客户结构剧变
