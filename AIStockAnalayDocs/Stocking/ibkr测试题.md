# IBKR 测试题知识点总结

## 一、股票仓位计算（Position）

**核心概念：**
- **Long（多头）**：持有股票，希望价格**上涨**（低买高卖）
- **Short（空头）**：借入并卖出股票，希望价格**下跌**（高卖低买，亏损无上限）

**仓位计算：**
```
持有量 − 卖出量 = 剩余仓位
+100股 − 100股 = 0股（Flat 持平）
```

| 操作       | 方向     | 盈利条件 |
| ---------- | -------- | -------- |
| Long 做多  | 低买高卖 | 希望涨   |
| Short 做空 | 高卖低买 | 希望跌   |

## 二、股息（Dividend）

- **定义**：公司将**利润的一部分**分配给**股东**的款项（公司 → 股东方向）
- **形式**：现金股息（cash）或股票股息（stock）
- **关键日期**：宣告日 → **除息日（Ex-Dividend）** → 登记日 → 支付日
- **领取条件**：必须在**除息日之前**持有股票

**除息日股价变化：**
> 除息日股价通常**下跌约等于每股股息**的金额（因为"领股息的权利"被剥离）

例：股价 $100，股息 $2 → 除息日约调整至 $98

## 三、订单类型（Order Types）

| 类型                    | 特点                 | 保证                     |
| ----------------------- | -------------------- | ------------------------ |
| **Market Order 市价单** | 立即按市场最优价成交 | 保证**成交**，不保证价格 |
| **Limit Order 限价单**  | 设定价格上/下限      | 保证**价格**，不保证成交 |

**限价单细节：**
- 买入限价单 = 指定愿付的**最高价**（成交价 ≤ 此价）
- 卖出限价单 = 指定愿接受的**最低价**（成交价 ≥ 此价）

---

## 四、保证金交易（Trading on Margin）

- **定义**：买股票时，部分金额由**券商（brokerage firm）贷款**支付
- **本质**：使用**杠杆（leverage）**放大购买力
- **vs 现金账户**：现金账户全额自有资金；保证金账户部分借贷

**风险提示：**
- 产生**保证金利息（margin interest）**
- 净值过低触发**追加保证金通知（Margin Call）**，可能被强制平仓
- 杠杆是**双刃剑**：放大收益也放大亏损

---

## 五、投资风险等级（Risk Levels）

**股票类型风险排序（低 → 高）：**

| 类型                       | 风险     | 说明                                     |
| -------------------------- | -------- | ---------------------------------------- |
| **Blue Chip 蓝筹股**       | 最低     | 大型稳健成熟公司                         |
| **Preferred Stock 优先股** | 较低     | 股息优先、清算受偿顺序高                 |
| **Common Stock 普通股**    | 中等     | 高于优先股                               |
| **Penny Stock 仙股**       | **最高** | 价格极低、易被操纵、流动性差、信息不透明 |

**大类对比：**
> **股票（Stocks）风险 > 货币市场产品（Money Market Products）**
> 货币市场产品（T-bills、商业票据、CDs）= 低风险、近似现金

## 六、盘后交易风险（Extended Hours Trading）

**依据：FINRA Rule 2265**

**属于盘后交易的风险（✅）：**
- 流动性更低（Lower liquidity）
- 波动性更高（Higher volatility）
- 价差更宽（Wider spreads）
- 价格不利、新闻公告风险、市场间不联通等

**不属于的（❌）：**
- **市场拥堵（Congested markets）** —— 盘后参与者少、市场冷清，"拥堵"反而是正常时段高峰特征

---

## 七、投资组合多元化（Diversification）

**正确做法：**
> 投资于**跨板块、跨行业的多种资产类别**，分散到不相关领域

**错误做法（不算多元化）：**
- ❌ 同一公司的优先股 + 普通股（仍是单一公司）
- ❌ 少数几只热门股票（集中度高）

**核心原则：**
> 不要把所有鸡蛋放在一个篮子里——**跨资产类别 + 跨行业 + 跨板块**
