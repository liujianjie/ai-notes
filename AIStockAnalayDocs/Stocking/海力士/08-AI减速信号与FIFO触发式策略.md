# 08 - AI 减速信号识别 + FIFO 400 股触发式策略

> 03/04 是"价格驱动"的 GTC 策略，本文是"信号驱动"的触发式策略。**两者并行使用** —— GTC 单赚牛市的钱，触发式策略避熊市的坑。

## 一、减速信号的"三级预警系统"

### 1.1 三级信号定义

| 级别 | 含义 | 操作模式 | 触发后窗口 |
|---|---|---|---|
| **🟡 黄色预警** | 单一指标转弱 | **观察 + 减仓 100 股** | 1-2 周内执行 |
| **🟠 橙色预警** | 两个指标共振 | **加速减仓 200 股 + 降低买回阈值** | 1 周内执行 |
| **🔴 红色预警** | 基本面拐点确认 | **强制清仓全部 + 切换 ADR** | 24-72 小时内执行 |

### 1.2 三类信号源 × 三级预警 矩阵

| 信号源 | 🟡 黄色（观察）| 🟠 橙色（加速）| 🔴 红色（清仓）|
|---|---|---|---|
| **AI 需求端** | Nvidia H100/B100 单季出货环比 -10% | 任一 Hyperscaler 下修 Capex ≥ 5% | **任一 Hyperscaler 下修 Capex ≥ 10%** 或 **连续 2 家下修** |
| **HBM 价格端** | HBM 现货价环比持平（停止上涨）| HBM 现货价环比 -3% | **HBM 现货价环比 -5%** |
| **SK Hynix 本身** | 韩本土单日 -8% 无明显利空 | 韩本土 1 周累计 -15% | **Q 财报 + "2027 订单"措辞下调** |
| **三星竞争** | HBM4 通过 Nvidia 认证 | HBM4 抢得 ≥ 20% Nvidia 配额 | **HBM4 抢得 ≥ 30% 配额** |
| **消费电子** | iPhone/MacBook YoY 持平 | YoY -5% | **YoY -10%+** |
| **KOSPI 系统** | 单周 1 次熔断 | 1 月 2 次熔断 | **1 月 3 次以上熔断** |

> **关键原则**：**橙色信号 = 2 个黄色共振，红色 = 1 个独立确认信号 OR 2 个橙色共振**。

## 三、FIFO 400 股的触发式策略（核心）

### 3.1 持仓状态

| Lot | 股数 | 成本 | FIFO 顺序 |
|---|---|---|---|
| Lot 1 | 100 | 90 | 队首 |
| Lot 2 | 100 | 90 | 第 2 |
| Lot 3 | 100 | 150 | 第 3 |
| Lot 4 | 100 | 170 | 队尾 |

### 3.2 触发式减仓决策表

| 当前预警级别 | 推荐操作 | FIFO 卖出 | 备注 |
|---|---|---|---|
| **正常**（无信号）| 维持原 03/04 GTC 单 | 按价格触发 | 保守 195 / 平衡 205 |
| **🟡 黄色（1 个信号）** | **降第一档卖价 -5 港元** | Lot 1 @ 190 (保守) / 200 (平衡) | 加速锁第一档 |
| **🟡 黄色（连续 2 周）** | **强制卖出 Lot 1（市价）** | Lot 1 (90) | 不等价格 |
| **🟠 橙色（2 个信号共振）** | **强制卖出 Lot 1 + Lot 2（市价）** | Lot 1 + Lot 2 | 24h 内 |
| **🟠 橙色 + 已触发 1 次** | **+ 卖出 Lot 3（市价）** | Lot 3 | 累计减仓 300 股 |
| **🔴 红色（基本面拐点）** | **强制清仓全部 400 股（市价）** | All | 不等盘中，下一个交易日完成 |
| **🔴 红色（叠加止损线 100）** | **市价清仓 + 切换 ADR 50% 资金** | All | 用剩余资金买 SK Hynix ADR |

### 3.3 触发式买回决策表

| 预警级别 | 是否允许买回 | 触发条件 |
|---|---|---|
| **正常**（无信号）| ✅ 允许 | 保守 -15% 深回调 / 平衡 -12% |
| **🟡 黄色** | ⚠️ 谨慎 | 仅在已减仓 ≥ 200 股 + 深回调 -20% + 双事件锚 |
| **🟠 橙色** | ❌ **禁止买回** | 信号不解除前不允许 |
| **🔴 红色** | ❌ **禁止买回 + 立即清仓所有 D 仓** | 含 D1/D2/D3 全部止损 |

### 3.4 信号解除条件

| 信号 | 解除条件 |
|---|---|
| 🟡 黄色 | 触发指标连续 2 周回到正常水平 |
| 🟠 橙色 | 两个触发指标 **同时** 回到正常水平 + 等 1 个月 |
| 🔴 红色 | **永久不解除** —— 一旦红色，本轮 07709 投资终止 |

## 五、最关键的 3 个判断节点（决策树）

### 5.1 节点 1：2026-07-25 Q2 财报

**判断价值**：决定"基本面有没有变" → 决定是否触发红色信号

**必看 4 个数据**：
1. **HBM ASP**（环比 / 同比涨幅）
2. **营业利润率**（是否突破 75%）
3. **"2027 订单已锁定"措辞**（是否仍出现）
4. **Q3 业绩指引**（是否上修）

**决策树**：
```
4 个数据全部超预期 → 维持原计划 GTC 单
1 个数据弱于预期 → 🟡 信号，降第一档卖价
2 个数据弱于预期 → 🟠 信号，市价卖 Lot 1 + Lot 2
"2027 订单"措辞软化 → 🔴 信号，市价清仓
```

### 5.2 节点 2：2026-10-25 Q3 财报

**判断价值**：HBM4 爬坡验证 + 三星竞争首次正面交锋

**必看 4 个数据**：
1. **HBM4 出货占比**（应 ≥ 25%）
2. **HBM4 良率**（应 ≥ 70%）
3. **三星 HBM4 通过 Nvidia 认证**（公开信息）
4. **2027 全年指引**（是否给）

**决策树**：
```
HBM4 出货占比 ≥ 30% + 良率 ≥ 75% → 牛市继续，维持原计划
HBM4 出货 < 25% 或良率 < 65% → 🟠 信号
三星抢得 Nvidia 配额 ≥ 20% → 🟠 信号
两个 🟠 共振 → 🔴 信号，立即清仓
```

### 5.3 节点 3：2027-01-25 Q4 财报 + 年度指引

**判断价值**：**全周期最关键的判断点** —— 决定是否启动"分段持有法切 ADR"

**必看 5 个数据**：
1. **2027 全年营收指引**（vs 共识）
2. **HBM4E 量产时间表**
3. **龙仁产能投产时间**
4. **资本支出指引**
5. **三星 / Micron 竞争格局**

**决策树**：
```
全年指引超共识 + HBM4E 准时 + 龙仁延期 → 🟢 维持持仓至 4/30
全年指引符合共识 → 🟡 加速减仓至 100 股
全年指引 < 共识 -5% → 🟠 立即减仓至 50 股
全年指引 < 共识 -10% → 🔴 全部清仓 + 切换 ADR 决策

特别：如果决定切换 ADR
→ 立即用清仓资金 70% 买 SK Hynix ADR
→ 30% 现金等 Q1 财报后二次买入
```

## 六、信号驱动 vs 价格驱动的优先级

**关键原则**：**信号优先于价格**。

| 情况 | 决策 |
|---|---|
| 价格在 195 但触发 🔴 红色信号 | **市价清仓**（不等价格触发 GTC）|
| 价格在 240 但无任何信号 | **维持 GTC，让它自然成交** |
| 价格在 150 但触发 🟠 橙色信号 | **市价清仓 200 股**（不等 145 防守线）|
| 价格在 92 但无信号 | **触发 100 保本止损单（按原计划）** |

**为什么信号优先？** 因为价格反应**滞后基本面 1-2 个季度**。看到信号还等价格触发 = 多承担 20-30% 跌幅。

## 八、Q&A

> **Q：怎么知道当前是几级预警？**
> A：每周日花 15 分钟，按本文 §1.2 的矩阵自评。**不会有人替你判断，必须主动监控**。

> **Q：如果黄色和橙色信号同时出现矛盾（如 HBM 涨价但 Nvidia 弱）？**
> A：**取保守者**。一个黄色 + 一个看多 = 仍按黄色处理。

> **Q：信号解除条件这么严格，会不会错过反弹？**
> A：**会，但这是保守策略的代价**。错过 20% 反弹 vs 损失 50% 下跌，选哪个？

> **Q：如果触发红色信号但价格还在 200+ 高位呢？**
> A：**仍然市价清仓**。基本面拐点确认后，价格反弹只是"末日狂欢"，不能恋战。

> **Q：07 这个文件和 06 有什么区别？**
> A：**06 讲"是否会进入周期"（宏观判断）**，**08 讲"AI 是否会萎靡 + 传导链"（深层因果）+ 触发式策略**。06 是仪表盘，08 是决策树。

## 九、最终建议

400 股 FIFO 持仓的最优策略组合：

1. **主轴**：执行 03（保守）或 04（平衡）的 GTC 单
2. **保险**：三级预警系统 + 触发式策略覆盖"基本面拐点"
3. **铁律**：2027-04-30 强制清仓 + 红色信号 24h 内清仓
4. **进阶**：CES + Q4 财报后判断是否切换 ADR（分段持有法）

> **核心心法**：**让 GTC 单帮你赚牛市的钱，让预警信号帮你避熊市的坑**。两者结合才是真正完整的策略。
